# qMarkdown

qMarkdown is a simple QT-based markdown viewer, it's light-weight and easy to use.

## Dependencies

* `qt5-base`

## Installation

### Make dependencies

* `g++` version 9 or above

### Arch based distros

There is an
[AUR package available](https://aur.archlinux.org/packages/qmarkdown/) for all
arch-based users

---

## Usage

`qmarkdown <file>`

### Keybindings

* `q` - Exits the program
* `g` - Go to TOP of file
* `G` - Go to BOTTOM of file
* `j` - Scroll DOWN
* `k` - Scroll UP
* `h` - Scroll LEFT
* `l` - Scroll RIGHT

#### WIP

* `o` - Opens textfield to enter filename of new file (WIP)
