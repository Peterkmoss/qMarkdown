## Usage

`qmarkdown [colorscheme] <file>`

### Colorschemes

* Light `-l` or `--light`
* Dark `-d` or `--dark`

### Keybindings

* `q` - Exits the program
* `g` - Go to TOP of file
* `G` - Go to BOTTOM of file
* `j` - Scroll DOWN
* `k` - Scroll UP
* `h` - Scroll LEFT
* `l` - Scroll RIGHT

#### WIP

* `o` - Opens a text field to enter filename of new file (WIP)
